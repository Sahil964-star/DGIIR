# DGIIR (Digital Grievance Intelligence & Incident Resolution)

## Git Workflow

Branch recommendations for the team of 4 developers:
- `main`: Production-ready code
- `frontend`: Frontend MVP development
- `backend`: Backend API development
- `ai`: AI classification and clustering modules
- `dashboard`: CM and Operations Dashboard development

## Environment Variables

Ensure the following variables are set before deployment or starting the server:

- `PORT`: Server port (default 5000)
- `MONGO_URI`: Database connection string
- `JWT_SECRET`: Secret key for access tokens
- `JWT_REFRESH_SECRET`: Secret key for refresh tokens (Required for startup)
